========================================
GLPI Keep Pending Status Plugin
========================================

DESCRIPTION:
Plugin for GLPI that keeps the "Pending" status on tickets when responses are added automatically.

VERSION: 1.0.0
AUTHOR: Gabriel Caetano
LICENSE: GPL v2 or higher
HOMEPAGE: https://github.com/gvcaetano190/keepPending

REQUIREMENTS:
- GLPI: 10.0.0 to 10.9.9
- PHP: 8.0 or higher
- Database: MySQL/MariaDB

INSTALLATION:
1. Download the plugin from: https://github.com/gvcaetano190/keepPending/releases/tag/v1.0.0
2. Extract to /path/to/glpi/plugins/keeppending
3. Access GLPI Administration > Plugins
4. Install KeepPending
5. Activate KeepPending

FEATURES:
- Keeps "Pending" status automatically
- Differentiates manual vs automatic status changes
- Complete logging system
- Easy installation via wget
- Multi-language support (Portuguese and English)
